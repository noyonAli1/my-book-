## 📖 Project Overview

Book Nest is a modern online book borrowing platform that digitizes the traditional library system. Users can explore a wide range of books, filter by categories, and borrow books easily. The application focuses on performance, security, and a smooth user experience using Next.js and BetterAuth.

---

## 🚀 Features

* 🔐 User Authentication (Email & Google Login)
* 📚 Browse All Books
* 🔎 Search Books by Title
* 📂 Category Filter (Story, Tech, Science)
* 📖 Book Details Page (Private Route)
* 📥 Borrow Book Feature
* 👤 User Profile with Update Option
* 📱 Fully Responsive (Mobile, Tablet, Desktop)
* ⚡ Fast Performance with Next.js

## 🔐 Environment Variables

Create a `.env.local` file and add:

MONGODB_URI=your_mongodb_uri
AUTH_SECRET=your_auth_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
